FBC_PACKAGE

Functions for using the SBML L3V1 fbc package.

Function are:

fbcEnabled = isBindFbcEnabled(path_to_fbc_file)

Takes

  1. path_to_fbc_file, the path to the fbc.xml file (optional)

Returns

  1. fbcEnabled =
    • 1 if the executables are enabled with fbc support
    • 0 otherwise

NOTE: if no argument is supplied it is assumed that the path is './test/test-data/fbc.xml'
