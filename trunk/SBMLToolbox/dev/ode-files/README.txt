The m file to be processed must be tagged with the tags
% SBML - begin init conds
% SBML - end init conds

% SBML - begin constants
% SBML - end constants

% SBML - begin ode function
% SBML - end ode function

See the file sample.m and sample_tagged.m for the tags.

Then run createSBMLModelFromFile(filename_of_tagged_m_file, output_sbml_filename)

which produces sampleSBML.
